# Hybrid-LSTM-GNN-model-for-stock-Price-Prediction
This repository contains three deep learning models implemented in PyTorch.

## Models Included
- Stack LSTM model for time-series prediction
- GCN model for graph-based learning
- Hybrid LSTM-GCN model combining temporal and graph features

## Requirements
- Python 3.8+
- PyTorch
- PyTorch Geometric
